# Calculadora CLI

Uma calculadora de linha de comando simples escrita em Python.

## Instalação

```bash
pip install -r requirements.txt
```

## Uso

```bash
python calculadora.py
```

## Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão

## Testes

```bash
python -m pytest tests/
```
