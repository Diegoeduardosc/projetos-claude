# CINEA — Dashboards & Ferramentas

Repositório de ferramentas e dashboards desenvolvidos para uso interno no CINEA / PUC Minas.

## Projetos

### CINEA BI Dashboard (`index.html`)
Dashboard de Business Intelligence para acompanhamento de projetos e iniciativas do centro. Interface web com dados em tempo real via Firebase Realtime Database. Acesso protegido por login.

**Funcionalidades:**
- Kanban de projetos por área
- Acompanhamento de status e progresso
- Modo de edição com salvamento automático no Firebase
- Notificações e badges de atualização
- Responsivo para desktop e mobile

### CINEA Dashboard (`CINEA_Dashboard.html`)
Versão complementar do painel com visualizações adicionais de acompanhamento.

## Tecnologia

- HTML + CSS + JavaScript puro (sem framework)
- Firebase Realtime Database (dados em tempo real)
- Google Fonts — DM Sans + Playfair Display
- Hospedado via GitHub Pages

## Acesso

O dashboard está disponível via GitHub Pages neste repositório.
